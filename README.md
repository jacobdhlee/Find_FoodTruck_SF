# Find food truck in SF

Find currently opened food truck list in San Francisco

## Getting Started

### Install

```
  npm i
```

and symlink

```
  npm link
```

### Running

```
  show-open-food-trucks
```

### Unlink

If you don't want to use anymore
```
npm unlink
```
